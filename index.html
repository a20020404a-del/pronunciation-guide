末尾子音
★★★★★
語末の子音 脱落注意
日本語は母音で終わる言語。英語の語末子音を省略すると全く伝わらない！
cat /t/ ✗ ca
bed /d/ ✗ be
【語末子音のコツ】 「ト」「ド」などを言わず 口の形だけ作って止める！ cat = キャッ[t口形] ↑「ト」とは言わない ↑口を閉じる形で終わる dog = ドッ[g口形] bed = ベッ[d口形] book = ブッ[k口形]
✅
コツ語末子音は「言う」ではなく「形を作る」！小さく聞こえてもOK。むしろ余計な母音（cat → キャット「ゥ」）を加えないことが大事。

★★★★★

有気音と無気音

英語のP/T/Kは息を強く出す「有気音」！

紙を使ったテスト ┌─────────────────────┐ │ 📄 ← 口の前に紙 │ │ │ │ P: 紙が揺れる → OK │ │ T: 紙が揺れる → OK │ │ K: 紙が揺れる → OK │ │ │ │ 揺れない → 練習必要！│ └─────────────────────┘

リンキング（音のつながり）

単語と単語をつなげると自然な英語に！

🎵 リズムとイントネーション

英語は「音楽」。リズムをつかむと劇的に自然になる！

🥁 英語は「強弱」のリズム言語

日本語は全部の音を均等に読む「モーラ言語」。英語は強い音と弱い音が交互に来る「ストレス言語」。

日本語 → 全部均等
わ・た・し・は・が・く・せ・い・で・す（全部同じ強さ）
英語 → 強弱がある
I am stu dent
← オレンジ = 強く　グレー = 弱く（速く）

🎢 イントネーション（音の上がり下がり）

同じ言葉でも「上がるか下がるか」で意味が変わる！

↘ 下降調 = 普通の文・断定
"I like coffee." ↘
"coffee" で音が下がる → 事実を伝えている
↗ 上昇調 = Yes/No質問・不確か
"Do you like coffee?" ↗
最後で音が上がる → Yes/Noを求めている
↗↘ 上昇→下降 = 列挙・驚き
"I want coffee ↗, tea ↗, and juice ↘."
リストの途中は上がって、最後だけ下がる
⚡ 内容語 vs 機能語

全部の単語を同じ強さで読まないのが英語のコツ！

🔊 内容語（強く読む）
名詞（book, cat）
動詞（run, think）
形容詞（big, hot）
副詞（very, fast）
🔇 機能語（弱く速く）
冠詞（a, an, the）
前置詞（in, on, at）
接続詞（and, but）
代名詞（I, you, he）
✅
例文で確認"I want to go to the store." → 強く読むのは "want", "go", "store" の3語だけ！
